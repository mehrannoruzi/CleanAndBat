# CleanArchitecture & BatFramework

An example of a Solution with a clean architecture along with integration with the Bat framework.

tools and frameworks used in this platform are :

1- .Net 7.x & C# 11
2- Entity Framework 7.x
3- Bat Framework
4- Quartz
5- DryIoc Container
6- EPPlus
7- MailKit
8- Swagger
