# HillaPay

This platform is a payment assistant.

in this platform use microsoft stack for back-end, 
and react js for front-end.

tools and frameworks used in this platform are :

1- .Net 7.0 & C# 11
2- Entity Framework 7.0
3- Bat Framework
4- Quartz
5- DryIoc Container
6- EPPlus
7- MailKit
8- Swagger
9- ReactJS
